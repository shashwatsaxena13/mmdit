# mmdit

This repository provides an implementation of **MMDIT** along with sample code written in **JAX**.

## Usage

To use the MMDIT JAX implementation, copy the repository into your project under the `third_party` directory.  
Ensure that the `pyytoml` dependency version is compatible with the requirements of your project.

## Installation

```bash
cd third_party/mmdit_jax
pip install -e .
